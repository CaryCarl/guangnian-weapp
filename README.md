# 📦 [光年视觉]

> 一个简洁、高效的微信小程序资源分享平台。致力于帮助用户快速发现、分享和管理优质学习/工具资源。

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/[你的GitHub用户名]/[仓库名]/pulls)

---

## ✨ 核心功能

* **🔍 资源检索**：支持按关键词、分类、标签快速搜索所需资源。
* **📁 资源分类**：涵盖素材、文档、工具、教程、链接等多种资源类型的精细化分类。
* **📤 快捷分享**：用户可一键发布和分享自己的珍藏资源。
* **⭐ 收藏**：支持收藏实用资源。
* **👤 个人中心**：管理我收藏的资源。
